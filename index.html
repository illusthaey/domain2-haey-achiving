기록관 업무보조(달력) 계산기

내 기록과 업무를 한 화면에서.

기록관은 날짜/태그로 정렬되고, 업무보조는 월간·주간·(오늘/내일) 2일 일정으로 운영합니다. 서버 없이 웹호스팅으로 먼저 돌리기 위해 데이터는 브라우저에 저장됩니다.

기록관
컨텐츠 감상 · 개발 학습 · 일상 로그를 날짜/태그로 조회
기능: 기록 추가, 태그 필터, JSON 내보내기/가져오기
업무보조
업무 일정 달력 (월간/주간/오늘·내일 2일)
기능: 일정 추가, 수정/삭제, 로컬 저장, JSON 백업
주의: localStorage 저장이라 같은 기기/브라우저에서만 유지됩니다. 다른 PC로 옮기려면 각 화면의 내보내기(JSON)로 백업 후 가져오기로 복원하세요.
